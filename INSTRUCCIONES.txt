INSTRUCCIONES PARA USAR LA APLICACIÓN DE CONTROL DE REFERIDOS
=====================================================

1. INICIAR LA APLICACIÓN
------------------------
- Haga doble clic en el archivo "start.bat"
- Se abrirá una ventana de comandos (no la cierre)
- El navegador se abrirá automáticamente con la aplicación
- Si el navegador no se abre, vaya a: http://localhost:3000

2. AGREGAR REFERIDOS
-------------------
- Escriba el nombre del trabajador
- Indique cuántos referidos cerró
- Haga clic en "Agregar"
- Si el trabajador ya existe, se sumarán los referidos

3. ELIMINAR REFERIDOS
--------------------
- Busque el trabajador en la tabla
- Haga clic en "Eliminar uno" para quitar un referido

4. EXPORTAR DATOS
----------------
- Haga clic en "Exportar a Excel"
- Se descargará un archivo CSV que puede abrir con Excel
- El archivo se guardará en su carpeta de Descargas

5. CAMBIO DE MES
---------------
- Cuando comience un nuevo mes, haga clic en "Reiniciar Datos"
- Se creará automáticamente una copia de seguridad con el nombre "backup_AÑO-MES.json"
- Esta copia se guarda en la misma carpeta de la aplicación

6. DÓNDE SE GUARDAN LOS DATOS
----------------------------
- Los datos se guardan automáticamente en el archivo "data.json"
- Las copias de seguridad se guardan como "backup_AÑO-MES.json"
- Todos los archivos están en la carpeta de la aplicación

7. CERRAR LA APLICACIÓN
---------------------
- Cierre el navegador cuando termine de usar la aplicación
- Cierre la ventana de comandos (puede hacer clic en "X" o escribir Ctrl+C)

IMPORTANTE: No elimine ni mueva los archivos de la carpeta de la aplicación
